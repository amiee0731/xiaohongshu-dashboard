# 小红书投放效果监测看板

交互式在线看板，用于监控小红书笔记投放效果。

## 访问链接
- 看板页面: https://amiee0731.github.io/xiaohongshu-dashboard/
- GitHub仓库: https://github.com/amiee0731/xiaohongshu-dashboard

## 功能特性
1. **四象限散点图**: CTR vs 进店率分析
2. **历史趋势对比**: 关键指标随时间变化
3. **双Tab数据表格**: 优质笔记与待优化笔记列表
4. **多维度筛选**: 按笔记类型、KOL等级、人群标签、时间范围筛选
5. **交互式hover**: 显示笔记详情与跳转链接

## 数据指标
- CTR (点击率): 阅读量 / 曝光量
- CPR (阅读成本): 达人合作金额 / 阅读量
- 互动率: 互动量 / 阅读量
- CPE (互动成本): 达人合作金额 / 互动量
- 进店率: 进店量 / 阅读量
- CPUV (进店成本): 达人合作金额 / 进店量

## 部署信息
- 部署方式: GitHub Pages
- 自动更新: 每日自动更新看板数据
